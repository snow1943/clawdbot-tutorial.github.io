安装教程

选择适合你的方式

不管你用什么设备，总有一种方式适合你 👇

安装前准备

GitHub 账号
云服务器（或本地电脑）
Node.js 18+
AI API Key（可选）
1

安装 Node.js（如果没有）

Clawdbot 需要 Node.js 运行环境，请根据你的系统选择安装方式：

🍎 Mac 用户
# 方式一：使用 Homebrew（推荐）
brew install node

# 方式二：直接下载安装包
# 访问 https://nodejs.org 下载 LTS 版本
🪟 Windows 用户
# 直接下载安装包
# 访问 https://nodejs.org 下载 LTS 版本
# 一路 Next 即可，记得勾选 "Add to PATH"
🐧 Linux 用户
# 使用包管理器
# Ubuntu/Debian:
sudo apt update
sudo apt install nodejs npm

# 或者使用 nvm（推荐，管理多版本更方便）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
✅ 验证安装
node -v   # 应该显示 v18.x.x 或更高
npm -v    # 应该显示版本号
2

克隆项目到本地

# 创建工作目录
mkdir -p ~/clawdbot && cd ~/clawdbot

# 克隆项目
git clone https://github.com/clawdbot/clawdbot.git .

# 切换到最新版本
git checkout main
3

安装依赖

# 使用 pnpm（更快）
npm install -g pnpm
pnpm install

# 或者使用 npm
npm install
💡 首次安装可能需要几分钟，耐心等待...
4

初始化配置

# 复制配置文件
cp config.example.yaml config.yaml

# 编辑配置文件（用你喜欢的编辑器）
nano config.yaml

# 或使用可视化配置（推荐新手）
npm run setup
📝 主要配置项：AI API Key、监听端口、启用哪些平台
5

启动服务

# 开发模式启动（支持热重载，改代码后自动重启）
npm run dev

# 生产模式启动
npm run build
npm start

# 🎉 看到 "Server started on http://localhost:3000" 就说明成功了！
6

访问管理后台

打开浏览器访问 http://localhost:3000，完成初始化配置。

🎉 恭喜！你已经成功搭建了 Clawdbot！接下来去配置平台接入吧。
1

安装 Docker（如未安装）

# Mac/Windows: 直接下载 Docker Desktop
# https://www.docker.com/products/docker-desktop/

# Linux (Ubuntu):
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 验证安装
docker --version  # 应显示 20.x.x 或更高
2

创建配置目录

# 创建数据目录
mkdir -p ~/clawdbot/{data,config}

# 进入目录
cd ~/clawdbot
3

创建配置文件

创建 config.yaml 文件，配置你的 AI API Key 等参数：

cat > config.yaml << 'EOF'
# Clawdbot 配置文件
server:
  host: "0.0.0.0"
  port: 3000

ai:
  provider: "openai"  # 可选: openai, anthropic, deepseek
  api_key: "你的API_KEY"

plugins:
  enabled:
    - weather
    - news
    - crypto
EOF
4

启动容器

# 启动 Clawdbot
docker run -d \
  --name clawdbot \
  -p 3000:3000 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/config:/app/config \
  -e NODE_ENV=production \
  clawdbot/clawdbot:latest

# 查看日志
docker logs -f clawdbot

# 🎉 访问 http://localhost:3000
5

更新版本

# 更新到最新版本
docker pull clawdbot/clawdbot:latest
docker restart clawdbot

💰 新用户有优惠 
# SSH 连接

# Docker Compose 方式
mkdir -p /opt/clawdbot
cd /opt/clawdbot

cat > docker-compose.yml << 'EOF'
version: '3.8'
services:
  clawdbot:
    image: clawdbot/clawdbot:latest
    restart: unless-stopped
    ports:
      - "3000:3000"
    volumes:
      - ./data:/app/data
EOF

docker compose up -d

使用场景 推荐配置 预估费用
个人试用 1核 2GB 50-100元/月
日常使用 2核 4GB 100-200元/月

1

使用 Helm 部署

# 添加 Helm 仓库
helm repo add clawdbot https://charts.clawd.bot
helm repo update

# 安装 Clawdbot
helm install clawdbot clawdbot/clawdbot \
  --namespace clawdbot \
  --create-namespace \
  --set replicaCount=2 \
  --set service.type=LoadBalancer
2

配置示例

# values.yaml 配置示例
replicaCount: 2

image:
  repository: clawdbot/clawdbot
  tag: latest

service:
  type: LoadBalancer
  port: 3000

ingress:
  enabled: true
  hosts:
    - host: clawdbot.example.com
      paths:
        - path: /
          pathType: Prefix

resources:
  requests:
    cpu: 500m
    memory: 512Mi
  limits:
    cpu: 2000m
    memory: 2Gi

📖 详细接入教程

  1. 打开 Telegram，搜索 @BotFather
  2. 发送 /newbot 创建新机器人
  3. 按照提示设置机器人名称和用户名
  4. 复制 Bot Token（格式：123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11）
  5. 在 Clawdbot 后台配置 Bot Token
  6. 设置 Webhook 或使用长轮询模式
  1. 登录飞书开放平台：https://open.feishu.cn/
  2. 创建企业应用，填写基本信息
  3. 在「凭证与权限」页面申请所需权限
  4. 获取 App ID 和 App Secret
  5. 在「事件订阅」配置回调 URL
  6. 在 Clawdbot 后台配置应用凭证
  1. 访问 https://api.slack.com/apps 创建新应用
  2. 选择「From scratch」创建应用
  3. 在「OAuth & Permissions」添加 Bot Token Scopes
  4. 安装应用到你的 Workspace
  5. 复制 Bot User OAuth Token
  6. 在「Event Subscriptions」启用并订阅消息事件
